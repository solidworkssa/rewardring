# RewardRing

Gamified loyalty reward system.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/RewardRing.sol`
- Stacks: `contracts/stacks/contracts/rewardring.clar`

## Setup
```bash
pnpm install
pnpm dev
```
